# Lake Time Brewery Website

This project was created for COM S 319 (Construction of User Interfaces) for Lake Time Brewery by Paige Rolling, Logan Ellsworth, and Tanner Smith using react and tailwind. 

## Pages Include:

- Home Page
- Beers Page
- About Page
- Merch Page
- Events Page
- Contact Page
